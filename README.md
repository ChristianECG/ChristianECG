### Proyectos en los que he trabajado recientemente

#### <a href="https://github.com/ChristianECG/TheBestCatPage">The best cat page</a>

<p align=justify>Como parte de las actividades del Hacktoberfest 2021, abrí un repositorio pensado para ser una página web que muestre gifs de gatos. Amistoso con quienes inician en el mundo open source, pero enfocado también en hacer una pequeña integración con Vue.js.</p>

<p align=justify>Integra HTML, CSS (Grid), Vue.js y Fetch.</p>

#### <a href="https://atomcloud.org">AtomCloud</a>

<p align=justify>El que puedo considerar como mi proyecto más ambicioso. AtomCloud es una solución general pensada para cualquier necesidad. Surge como un recurso que centralizara diversas herramientas a través de su API, y se ha convertido en mi pasatiempo de desarrollo favorito.</p>

<p align=justify>Su core está hecho con Vue.js, Axios, PHP y MySQL. Además, incorpora Bootstrap.</p>

#### <a href="https://itsoeh.dev/">Developer Student Club ITSOEH</a>

<p align=justify>Este proyecto fue creado con mucho amor para la comunidad del Developer Student Club ITSOEH. Esta comunidad está orientada a jóvenes estudiantes interesados en el desarrollo de software, de la mano de Google.</p>

<p align=justify>El sitio está desarrollado con PHP, HTML, CSS y Bootstrap. Además, incorpora llamadas a la API propia para el tema de eventos.</p>
